This repository is a pursuit of Advent of Code 2020 challenge written in python3, displaying the results (strictly linked to my file inputs) and stars as a passed day.

Type in your terminal below commands to run all script at once, then second command to see the results.

```sh
$ ./run.sh
$ nano ./output/aoc.txt
```
